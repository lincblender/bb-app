# BidBlender Golden Path Narration

Use this with `npm run video:record` in guided mode.

## 1. Sign in

Say:
Start with a real workspace account so the walkthrough opens on live data instead of a canned shell.

## 2. Open a clean chat

Say:
I reset to a fresh thread so the recording starts from a clean qualification path, even if the tenant already has older chats.

## 3. Ask for matching bids

Say:
This prompt resolves from tenant data first. The product should be able to find matching work without pretending a language model is a search engine.

## 4. Focus one opportunity

Say:
Now I turn the broad result set into a single qualification thread around the Whole-of-Government Cloud Framework.

## 5. Run SWOT from context

Say:
Once the opportunity is in context, the network panel already holds the shape of the SWOT. That makes this a fast, structured check rather than another blank-prompt exercise.

## 6. Ask competitor impact

Say:
This is the higher-value move. I am not just checking fit, I am asking what competitor behavior does to our position and what action we should take next.

## 7. Open the full opportunity detail page

Say:
Chat is the fast front door, but the detail page is where the score breakdown, issuer context, and next actions become easy to inspect.

## 8. Return to chat for document review

Say:
The final process is document review. I want to show extraction first, then the review response, while staying honest about what the system actually analysed.

## 9. Upload a sample brief and run review

Say:
If live AI analysis is configured, this can produce a fuller review. If not, the workflow still demonstrates extraction and the explicit fallback guardrail.
